En este caso, la variable `x` representa una cantidad desconocida pero de la que se sabe si se le suma 3 obtiene 4. Resolviendo la ecuaci√≥n, entendemos inmediatamente que la variable `x` estaba representando realmente el n√∫mero 1.

## **Variables Globales y Locales**

En Python, una variable puede ser **global** o **local**. Las variables globales son accesibles desde cualquier parte del c√≥digo, mientras que las variables locales solo son accesibles dentro de una funci√≥n o bloque de c√≥digo en el que se definen.

> Los ejemplos son para demostrar la diferencias entre variables globales y locales, ya pronto veremos m√°s sobre las **funciones**.

```python

def mi_funcion():
    variable_local = "Soy local"

    print(variable_local)   # Accede a la variable local

mi_funcion()

```
